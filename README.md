# R3F Chibi Shootgame

![](./img/title.png)

This is a demonstration to create a little game with libraries under below:

- R3F (Three.js + React)
- Dragonbones
- Zustand

## Features

- Mouse aiming
- Keyborde control
- Skeletal Animation (Dragonbones)
- BGM and simple sound effect
- Enemy
  - Move
  - Attack
  - Destroyable
- HUD (Minimap)
- Map boundary
